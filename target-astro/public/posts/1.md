# Getting Started with Nextcloud on Unraid

> A step-by-step guide to self-hosting your own cloud storage at home.

![Nextcloud on Unraid](https://images.unsplash.com/photo-1558494949-ef010cbdcc31?w=800)

## Why Self-Host?

When you use Google Drive or Dropbox, your data lives on someone else's server. Self-hosting Nextcloud means **your data stays with you** — encrypted, private, and fully under your control.

## Prerequisites

- An Unraid server with at least 8GB RAM
- A domain name with DNS pointing to your home IP
- Cloudflare for tunneling (no open ports required)

## Step 1: Install the Nextcloud Docker Container

Navigate to the **Apps** tab in your Unraid UI and search for "Nextcloud". Select the official Linuxserver.io image for best compatibility.

## Step 2: Configure Cloudflare Tunnel

Instead of opening port 443, use a Cloudflare Tunnel to expose Nextcloud securely. This hides your home IP and adds DDoS protection for free.

```bash
cloudflared tunnel create nextcloud
cloudflared tunnel route dns nextcloud cloud.yourdomain.com
```

## Step 3: Enable Redis for Performance

Add a Redis container and configure Nextcloud to use it as a memory cache. This dramatically improves file listing and sync speeds.

That's it! You now have a private, self-hosted cloud storage solution running on your own hardware.
